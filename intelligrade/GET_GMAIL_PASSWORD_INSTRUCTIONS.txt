After getting your Gmail App Password from Google:
1. Copy the 16-character password (like: abcd efgh ijkl mnop)
2. Open backend/.env file
3. Replace "ADD_YOUR_GMAIL_APP_PASSWORD_HERE" with your actual password

Example:
SMTP_PASSWORD=abcd efgh ijkl mnop

Then restart the backend server:
cd backend && python -m backend.main
